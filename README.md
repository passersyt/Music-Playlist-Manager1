# 🎵 Music Playlist Manager

A console-based Music Playlist Manager developed in **C++** using **Doubly Linked List**.

## Features
- Add Song
- Display Playlist
- Play Song
- Next / Previous Song
- Search Song
- Update Song
- Delete Song
- Favourite Songs
- Shuffle Playlist
- Sort Playlist
- Save & Load Playlist
- Playlist Statistics

## Data Structure Used
- Doubly Linked List

## Technologies
- C++
- Object-Oriented Programming (OOP)
- File Handling

## Compile

```bash
g++ -std=c++17 main.cpp Playlist.cpp Song.cpp -o MusicPlayer
```

## Run

```bash
./MusicPlayer
```
